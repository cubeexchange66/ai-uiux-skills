# AI UI/UX Design Skills (Global Standard)

This repository contains the definitive "Yin & Yang" design system for AI agents, optimized for creating premium, non-generic user interfaces.

## The Core System

The system consists of **three integrated modules** that work together:

### 1. The Builder: UI/UX Design Skill (`uiux-design-skill.md`)
**"What TO Do"**
- Implements deep research principles: Kinetic Physics, Cognitive Color Theory, and Spatial Geometry.
- Enforces the **8-Point Grid** system for perfect spacing.
- Uses the **60-30-10 Rule** for color harmony.
- **Goal:** Create scientifically beautiful interfaces.

### 2. The Guardian: Uncodixify (`uncodixfy.md`)
**"What NOT To Do"**
- A strict firewall against "Generic AI Design" (Codex UI).
- Bans specific clichés: excessive glassmorphism, weird gradients, and pill-shaped buttons everywhere.
- Enforces "Normal UI" standards inspired by Linear, Raycast, Stripe, and GitHub.
- **Goal:** Prevent the robotic "AI Look".

### 3. The Lab: Design Laboratory (`LABORATORY.md` + `MUTATION_PROTOCOL.md`)
**"Where Innovation Happens"**
- An R&D engine for experimental design.
- Uses a genetic algorithm (Mutation Protocol) to tweak physics, typography, and geometry.
- Automatically generates 3 variants for every request:
    - **Variant A (Safe):** Standard & Reliable.
    - **Variant B (Evolution):** Modern Polish (+10% Magic).
    - **Variant C (Mutation):** First-Mover Innovation.
- **Goal:** Create unique, trend-setting designs autonomously.

## The Library (`library/`)
Successful experiments from The Lab are stored here as reusable React components.
- `library/standard/`: Production-ready assets.
- `library/experimental/`: Avant-garde concepts.
- `library/atoms/`, `library/molecules/`, `library/organisms/`: Atomic design structure.

## Usage

To use these skills in your AI agent (Copilot CLI, etc.):

1.  Copy the `.md` files to your agent's skill directory (e.g., `.copilot/skills/`).
2.  **Standard Mode:** "Agent, create a button." (Uses Builder + Guardian)
3.  **Lab Mode:** "Agent, **experiment** with a button." (Uses Lab + Mutation Protocol)

## Philosophy

> "Design is not just what it looks like and feels like. Design is how it works." — Steve Jobs

This system moves away from "creative guessing" towards **engineered design decisions** based on physics and psychology.
