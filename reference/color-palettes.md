# Sophisticated Color Palettes - Anti-AI Collection

## 🚫 BANNED COLORS (Never Use)

These colors scream "AI-generated" and must be avoided:

### Primary Colors - FORBIDDEN:
- **Blue**: `#0066FF`, `#3B82F6`, `#2563EB` (Tailwind blue defaults)
- **Purple/Violet**: `#8B5CF6`, `#A855F7`, `#7C3AED` (AI favorite)
- **Pink**: `#EC4899`, `#DB2777` (too obvious)
- **Cyan**: `#06B6D4`, `#0891B2` (overused everywhere)
- **Indigo**: `#6366F1`, `#4F46E5` (generic SaaS)

### Why These Are Banned:
1. Most common in AI-generated UI
2. Tailwind defaults that everyone uses
3. Zero personality or uniqueness
4. Users instantly recognize "this is AI-made"

## 💡 PRO TIPS

### 1. Use RGB Values AI Doesn't Know
```css
/* ❌ Generic */
--primary: #3B82F6;  /* Tailwind blue-500 */

/* ✅ Unique */
--primary: #3D5A6C;  /* Custom ink blue */
```

### 2. Muted > Bright
- Bright colors scream AI
- Muted colors feel curated and professional

### 3. Add Brown/Earth Tones
- AI rarely uses browns
- Browns = sophisticated & grounded

### 4. Single Accent Only
- Not a rainbow of colors
- One bold accent creates personality

### 5. Test in Grayscale
- If hierarchy still works without color
- Colors are functional, not decorative

## 📦 IMPLEMENTATION EXAMPLE

```tsx
// Real-world usage: E-commerce site

export const colors = {
  // Terracotta & Sage palette
  brand: {
    primary: '#E0624B',      // Terracotta
    secondary: '#6B8E6F',    // Sage
    accent: '#C9A362',       // Ochre
  },
  
  background: {
    canvas: '#FAFAF9',       // Warm white
    surface: '#FFFFFF',      // Pure white
    subtle: '#FAF7F2',       // Hover states
  },
  
  text: {
    primary: '#2C2925',      // Warm black
    secondary: '#57534E',    // Stone-600
    tertiary: '#78716C',     // Stone-500
    disabled: '#A8A29E',     // Stone-400
  },
  
  border: {
    default: '#E7E5E4',      // Stone-200
    hover: '#D6D3D1',        // Stone-300
    strong: '#A8A29E',       // Stone-400
  },
}

// Usage in components
<button style={{ 
  backgroundColor: colors.brand.primary,
  color: 'white'
}}>
  Add to Cart
</button>
```
